# Welcome to the example vault

This is a tiny vault you can use to try out CMA without setting up your own.

Run from the repo root:

```
cma index examples/example_vault/..
cma graph health examples/example_vault/..
```

See [[Async Capital Call Processing]] for an example decision note linked to
[[Queue Retry Pattern]] and [[External API Synchronous Bottleneck]].
